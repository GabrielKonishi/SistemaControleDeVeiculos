# SistemaControleDeVeiculos

### Sistema de controle de veiculos desenvolvidos para estudos integrando front end ao backend

## campos mapeados no banco de dados
+ veiculo;
+ marca;
+ ano;
+ descricao;
+ disponibilidade;
+ dataCadastro;
+ dataAtualizacao;

## ferramentas utilizadas
+ Spring Boot
+ Spring JPA
+ Spring Web
+ Banco de dados MySql
